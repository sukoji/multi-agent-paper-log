## 한 줄 요약

완전 오픈 파이프라인으로 훈련된 bilingual LLM—agent backbone 후보로 스터디.

## 배경 · 문제 정의

Agent 연구가 GPT-4에만 의존하면 재현이 안 된다. MAP-Neo는 data·code·weight를 공개해 중형 모델로 agent 실험 가능하게 한다.

## 핵심 방법

- End-to-end open training pipeline documentation
- 7B bilingual pretrain + SFT + alignment stages
- Transparent data mixture reporting
- Benchmark suite on EN/ZH tasks

## 실험 · 결과

- 동급 open model 대비 competitive MMLU·C-Eval
- Full toolchain for reproduction
- Suitable as local agent LLM backbone

## 한계 · 비판적으로 본 점

Tool use·long context는 frontier model 대비 약함. Agent-specific fine-tune은 사용자 몫.
