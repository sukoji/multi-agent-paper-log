## 한 줄 요약

Minecraft에서 skill library + automatic curriculum로 평생 학습하는 embodied LLM agent.

## 배경 · 문제 정의

기존 RL agent는 task마다 재학습해야 한다. Voyager는 성공한 코드를 skill로 저장하고 점점 어려운 goal을 스스로 제안한다.

## 핵심 방법

- Automatic Curriculum이 다음 탐험 목표 제안
- Skill Library에 검증된 JS mineflayer 코드 저장·retrieval
- Iterative Prompting with environment feedback and execution errors
- GPT-4 code generation + embedding search

## 실험 · 결과

- Prior SOTA 대비 3.3× 더 많은 unique item 획득
- Tech tree 전 범위에서 skill transfer
- open-ended exploration에서 novelty 유지

## 한계 · 비판적으로 본 점

GPT-4 의존도가 높고 실제 물리 로봇 이식은 안 됨. Skill library versioning이 eval에 빠져 있음.

## TIL — 내가 가져간 점

self-evolving agent taxonomy에서 'skill library growth curve'를 표준 메트릭으로 넣고 싶다.
