## 한 줄 요약

API document retrieval + fine-tuned LLM으로 hallucinated API call을 줄임.

## 배경 · 문제 정의

LLM이 존재하지 않는 API를 지어내는 문제가 심각하다. Gorilla는 retriever로 doc을 붙이고 APIBench로 학습한다.

## 핵심 방법

- Retriever fetches relevant API docs at inference
- Fine-tune on APIBench: TorchHub, TensorHub, HuggingFace APIs
- AST matching for call correctness evaluation
- Gorilla OpenFunctions evolution for JSON schema calls

## 실험 · 결과

- API argument accuracy GPT-4 대비 우수 (domain-specific)
- Retriever reduces hallucinated endpoints
- OpenFunctions v2 multi-tool support

## 한계 · 비판적으로 본 점

API catalog update lag. Multi-step dependency between calls 미약.
