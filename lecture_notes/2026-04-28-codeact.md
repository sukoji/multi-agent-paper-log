## 한 줄 요약

JSON tool 대신 Python code로 action을 표현해 agent expressiveness 향상.

## 배경 · 문제 정의

Fixed JSON schema는 복잡한 data transform에 부족하다. CodeAct는 unified code execution으로 tool use를 일반화한다.

## 핵심 방법

- Agent generates Python with API bindings in sandbox
- Multi-turn: code output feeds next turn
- Fine-tune CodeActInstruct dataset
- Compare JSON-tool vs code on API-Bank, M3ToolEval

## 실험 · 결과

- Success rate up to 20%p on complex API tasks
- Code reuse across steps efficient
- OpenHands default agent family

## 한계 · 비판적으로 본 점

Sandbox escape risk. Debugging generated code adds latency.
