## 한 줄 요약

Role·goal·backstory를 가진 agent crew가 sequential/hierarchical process로 협업.

## 배경 · 문제 정의

LangChain만으로는 팀 역할과 delegation이 boilerplate가 많다. CrewAI는 'crew' abstraction으로 enterprise workflow를 빠르게 조립한다.

## 핵심 방법

- Agent 정의: role, goal, tools, LLM
- Task object에 expected output과 context chain
- Process: sequential, hierarchical (manager-led)
- Crew kickoff으로 end-to-end run

## 실험 · 결과

- 마케팅·research crew 예제 다수
- hierarchical process에서 manager delegation
- 빠른 프로토타이핑 community adoption

## 한계 · 비판적으로 본 점

Deep observability와 deterministic replay가 약함. Manager agent 비용이 crew 전체를 지배.
