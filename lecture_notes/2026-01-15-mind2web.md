## 한 줄 요약

2000+ real website에서 general web action prediction dataset.

## 배경 · 문제 정의

기존 web dataset은 소수 template에 overfit된다. Mind2Web는 diverse real site crawl로 generalization을 측정한다.

## 핵심 방법

- Crowdsourced tasks on 137 websites, 31 domains
- Action representation: element op + value
- Candidate generation + action selection 두 단계
- Cross-task, cross-website, cross-domain splits

## 실험 · 결과

- Cross-domain generalization이 핵심 난제로 확인
- LLM fine-tune이 prior heuristic보다 우수
- Element retrieval error가 bottleneck

## 한계 · 비판적으로 본 점

Offline dataset—live execution feedback 없음. Dynamic SPA 변경에 취약.
