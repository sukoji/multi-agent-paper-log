## 한 줄 요약

LLM client와 tool/data server 간 open protocol 표준.

## 배경 · 문제 정의

M개 모델 × N개 tool integration이 반복된다. MCP는 JSON-RPC 기반 capability discovery를 표준화한다.

## 핵심 방법

- MCP server exposes tools, resources, prompts
- Client discovers via capability handshake
- stdio/SSE transport
- Reference SDK TypeScript/Python

## 실험 · 결과

- Claude Desktop, Cursor, Zed adoption
- Growing server directory (git, slack, db)
- Decouples tool auth from model provider

## 한계 · 비판적으로 본 점

Security model still maturing. No multi-agent orchestration layer.
