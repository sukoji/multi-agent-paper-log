## 한 줄 요약

ToolBench dataset와 ToolLLaMA로 대규모 real API tool use 학습.

## 배경 · 문제 정의

소수 API 예제로는 general tool mastery가 안 된다. ToolLLM은 RapidAPI 16k+ API instruction data를 구축한다.

## 핵심 방법

- ToolBench: ChatGPT-assisted trajectory generation on real APIs
- Depth-first search decision tree for API chain exploration
- ToolLLaMA fine-tune on collected paths
- ToolEval automatic evaluator

## 실험 · 결과

- ToolLLaMA가 ChatGPT tool use와 유사한 성능 (일부 subset)
- Multi-API chain solving 가능
- Open ToolBench dataset 공개

## 한계 · 비판적으로 본 점

API key·rate limit으로 eval 재현 어려움. Trajectory noise 존재.
