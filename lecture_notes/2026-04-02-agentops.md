## 한 줄 요약

Agent session recording, cost, latency, failure tagging SaaS.

## 배경 · 문제 정의

Production agent는 prompt dump만으로는 debug 불가. AgentOps는 span-like agent event를 수집한다.

## 핵심 방법

- SDK decorator on LLM and tool calls
- Session replay with tree view
- Cost and token aggregation
- Exception and hang detection

## 실험 · 결과

- Integration with CrewAI, AutoGen, LangChain
- Dashboard for session drill-down
- Alert on cost anomaly

## 한계 · 비판적으로 본 점

Vendor lock-in. PII in traces needs redaction policy.
